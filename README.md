# QTtt (Custom Tic-Tac-Toe) 🎮

A versatile implementation of the classic Tic-Tac-Toe game, built with **C++** and **Qt 6 (QML)**. Forget the standard 3x3 grid — here, you define the rules yourself!

## ⚠️ Project Status: Alpha
This project is currently **under active development**. 
Features may be limited, and you might encounter bugs. We are working towards a stable Beta release.

## Features
- 🧩 **Customizable Grid**: Set any grid size you want (Width x Height).
- 👥 **Multiplayer Mode**: Play with two players or add more participants.
- 🖼️ **Custom Skins**: Use any image for each player instead of the boring `X` and `O`.
- 🏆 **Win Conditions**: Define the required line length (win length) to secure a victory.
- 📱 **Cross-Platform**: Support for Desktop (Linux, Windows) and Android.

## Technologies
- **C++17** & **Qt 6.2.4**
- **QML** for a modern and adaptive user interface.
- **CMake** as the primary build system.
- **GitHub Actions** for automated CI/CD across all platforms.
