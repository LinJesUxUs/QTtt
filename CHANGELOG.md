# Changelog

## [0.1.0-alpha] - 2026-03-28

#### Initial release.

### Features
- Customizable grid sizes with support for non-square grids
- Multiple player support (2+ players)
- Custom player skins using any image format
- Flexible win conditions (configurable line length)
- Cross-platform support: Desktop (Linux) and Android
- Modern UI built with Qt 6.2.4 and QML
- Automated CI/CD pipeline with GitHub Actions
- Comprehensive unit tests for core game logic

### Known Issues
- Some features may not work as expected.
- Performance may degrade with larger data sets.
